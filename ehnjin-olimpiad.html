Pseudocode гарын авлага

FOR / WHILE / ARRAY — 7-р ангийн олимпиадын зөөлөн бэлтгэл

Зорилго: ойлгох → бодлого
Хэрэгсэл: дэвтэр ✏️
Хариу түлхүүр: байхгүй
Хугацаа: 40–60 мин

Доторх хэсгүүд (товшоод очно)

  • 0) Pseudocode гэж юу вэ?
  • 1) Хувьсагч (variable)
  • 2) INPUT / PRINT
  • 3) IF нөхцөл
  • 4) FOR давталт + STEP
  • 5) WHILE давталт
  • 6) ARRAY (массив)
  • 7) Шалгах хуудас
  • 8) Дасгал (дэвтэр дээр)

0) Pseudocode гэж юу вэ?

“Код биш, гэхдээ код шиг” бичдэг алхамчилсан заавар.

Pseudocode = хүний хэлтэй ойр, гэхдээ мөр мөрөөр тодорхой бичдэг заавар.

Яагаад хэрэгтэй вэ?
Эхлээд “юу хийх вэ?” гэдгийг ойлгоно → дараа нь Python/C руу амархан шилжинэ.

Гол дүрэм (маш энгийн)

  1. Нэг мөр = нэг үйлдэл
  2. Дотогшоо зай авах (indent) = “энэ нь дотор нь” гэсэн үг
  3. Том/жижиг үсэг хамаагүй (гэхдээ тогтвортой бичвэл гоё)

1) Хувьсагч (variable)

Тоон утгыг хадгалах “хайрцаг”.

= тэмдэг нь “тэнцүү” биш. Энэ нь утга оноож байна гэсэн санаа.

x = 5
y = 10
sum = x + y
PRINT sum

2) INPUT / PRINT

INPUT = хэрэглэгчээс авна, PRINT = дэлгэцэнд гаргана.

INPUT (авах)

INPUT n

“n гэсэн хувьсагч руу хэрэглэгчийн оруулсан утгыг хий.”

PRINT (гаргах)

PRINT n
PRINT "Hello"

“Дэлгэцэнд харуул.”

3) IF нөхцөл

Нөхцөл үнэн/худал эсэхээс хамаарч өөр өөр үйлдэл хийнэ.

Бүтэц:

IF нөхцөл THEN
    хийх зүйл
ELSE
    өөр хийх зүйл
END IF

Жишээ: n тоо эерэг үү?

INPUT n
IF n > 0 THEN
    PRINT "positive"
ELSE
    PRINT "not positive"
END IF

4) FOR давталт + STEP

FOR = тоолон давтах. STEP = хэдээр нэмэгдэх/бууруулах вэ?

FOR нь “тоо тоолж давт” гэсэн санаа.
Жишээ нь: 1-ээс 10 хүртэл, эсвэл 10-аас 1 хүртэл.

4.1 Энгийн FOR (алхам 1)

FOR i = 1 TO 5
    PRINT i
END FOR

Гаралт: 1 2 3 4 5

4.2 STEP (алхам) тохируулах

1-ээс 10 хүртэл 2-оор нэмэгдүүл:

FOR i = 1 TO 10 STEP 2
    PRINT i
END FOR

Гаралт: 1 3 5 7 9

10-аас 1 хүртэл бууруул (reverse):

FOR i = 10 TO 1 STEP -1
    PRINT i
END FOR

4.3 FOR дотор “юу болж байна?” (алхамчилж ойлгох)

FOR i = 1 TO 4
    PRINT i
END FOR
  • i = 1 → хэвлэнэ
  • i = 2 → хэвлэнэ
  • i = 3 → хэвлэнэ
  • i = 4 → хэвлэнэ
  • i = 5 болох гэж оролдоно → 4-өөс хэтэрсэн тул STOP

5) WHILE давталт

WHILE = нөхцөл үнэн үед давт.

Бүтэц:

WHILE нөхцөл
    хийх зүйл
END WHILE

Жишээ: 1-ээс 5 хүртэл хэвлэх

i = 1
WHILE i <= 5
    PRINT i
    i = i + 1
END WHILE
Анхаар: WHILE дотор тоолуурыг өөрчлөх мөр байхгүй бол давталт зогсохгүй!

Жишээ: хэрэглэгч 0 оруулах хүртэл нэмж яв

sum = 0
INPUT x
WHILE x != 0
    sum = sum + x
    INPUT x
END WHILE
PRINT sum

6) ARRAY (массив)

Олон утгыг нэг нэр дор хадгалах “тавцан”. Индексээр хандана.

ARRAY = олон тоо хадгална.
a[1], a[2] гэх мэтээр дугаараар нь авна.

6.1 Зарлах (declare)

DECLARE a[5]

“a гэдэг нэртэй 5 ширхэг утга хадгална.”

Санамж: Зарим бодлогод индекс 0-ээс эхэлдэг. Энэ гарын авлагад ойлгоход амар гэж 1-ээс эхлүүлж бичлээ.

6.2 Утга оноох (гараар)

a[1] = 10
a[2] = 7
a[3] = 9
a[4] = 3
a[5] = 8

6.3 FOR ашиглаад array унших

DECLARE a[5]
FOR i = 1 TO 5
    INPUT a[i]
END FOR

6.4 Нийлбэр (sum)

DECLARE a[5]
FOR i = 1 TO 5
    INPUT a[i]
END FOR

sum = 0
FOR i = 1 TO 5
    sum = sum + a[i]
END FOR

PRINT sum

6.5 Хамгийн их (max)

DECLARE a[5]
FOR i = 1 TO 5
    INPUT a[i]
END FOR

max = a[1]
FOR i = 2 TO 5
    IF a[i] > max THEN
        max = a[i]
    END IF
END FOR

PRINT max

6.6 Хамгийн бага (min)

min = a[1]
FOR i = 2 TO 5
    IF a[i] < min THEN
        min = a[i]
    END IF
END FOR

PRINT min

7) Шалгах хуудас (алдаа гарахаас сэргийлэх)

Бодлого бодохдоо “энэ 3-ыг” үргэлж шалга.

FOR дээр

  • Эхлэх утга зөв үү?
  • Төгсгөлийн утга зөв үү?
  • STEP зөв үү?
  • Индекс/тоолуур хэтрэхгүй юу?

WHILE дээр

  • Нөхцөл нэг өдөр худлаа болох уу?
  • Дотор нь тоолуур/утга өөрчлөгдөж байна уу?
  • STOP хийх боломж байна уу?

ARRAY дээр

  • Индекс хэтрэхгүй юу? (1..N)
  • max/min эхлүүлэхдээ a[1]-ийг зөв ашигласан уу?
  • sum-аа 0-оос эхлүүлсэн үү?

8) Дасгал (дэвтэр дээр хий)

Хариу түлхүүр байхгүй. Өөрөө бодоод, шалгуулна.

Дасгал A — FOR + STEP

  1. 1-ээс 20 хүртэл 3-аар нэмэгдүүлж хэвлэ
  2. 50-аас 10 хүртэл 5-аар бууруулж хэвлэ

Дасгал B — WHILE

  1. 1-ээс эхлээд 2-оор нэмсээр 20-оос хэтрэх хүртэл хэвлэ
  2. Хэрэглэгч -1 оруулах хүртэл тоонуудыг уншаад нийлбэрийг гарга

Дасгал C — ARRAY

  1. N тоо уншаад нийлбэр, max, min гарга
  2. N тоо дундаас 0 хэд байгааг тоол
Зөвлөгөө: Дасгалаа хийхдээ эхлээд pseudocode-оо “ярианы хэлээр” нэг өгүүлбэрээр тайлбарлаад, дараа нь мөр мөрөөр нь бичээрэй.
Энэ хуудас нь утсаар уншихад тохиромжтой (mobile friendly).
Хүсвэл дараагийн хуудсыг “1-р бүлэг: Array бодлогууд (10 бодлого)” гэж үргэлжлүүлж өгч болно.
