Variable & Pseudocode Basics

Хүүхэд өөрөө уншаад, дэвтэр дээрээ ажиллаж сурах хуудас

1. Variable (Хувьсагч)

Variable гэдэг нь мэдээлэл хадгалах нэртэй хайрцаг юм.

  • name – нэр хадгалах
  • age – нас хадгалах
  • sum – нийлбэр хадгалах
Санах: Хайрцаг бүрт ойлгомжтой нэр өгвөл програм уншихад амархан.

2. INPUT – ASSIGNMENT – OUTPUT

INPUT – хэрэглэгчээс мэдээлэл авах

INPUT age INPUT name

ASSIGNMENT – утга өгөх, бодох

x = 5 y = x + 1

OUTPUT (PRINT) – хариуг дэлгэцэнд гаргах

PRINT x PRINT "Hello"
"=" тэмдэг нь тэнцүү биш. Баруун талын утгыг зүүн талын хувьсагчид хийж байна.

3. Pseudocode basics

Pseudocode бол компьютерийн хэл биш. Хүн ойлгохоор алхамчилж бичсэн заавар.

START INPUT a INPUT b sum = a + b PRINT sum END
Ихэнх бодлого дараах дарааллаар явна: INPUT → ASSIGNMENT → OUTPUT

Дэвтэр дээр хийх даалгавар

Даалгавар 1
Доорх pseudocode-г дэвтэр дээрээ хуулж бич.
START INPUT name PRINT "Сайн уу, " + name END

• Компьютер юуг асууж байна вэ?
• Дэлгэцэнд юу гарч ирэх вэ?

Даалгавар 2
Дараах кодыг дэвтэр дээр бичээд эцсийн утгыг ол.
x = 10 y = x + 5 x = y + 2

• y хэд болсон бэ?
• x хэд болсон бэ?

Даалгавар 3
Дараах бодлогын pseudocode-г өөрөө бич.

Нэг тоо INPUT авна. 2-оор үржүүлнэ. Хариуг PRINT хийнэ.

Дасгал бодлогууд (Бие даан ажиллах)

Бодлого 1
Компьютер хэрэглэгчээс нэг утга асууна.
Тэр утгыг 3-аар ихэсгэнэ.
Гарсан үр дүнг дэлгэцэнд харуулна.

➜ Алхамчилж бодоод pseudocode хэлбэрээр дэвтэр дээрээ бич.

Бодлого 2
Доорх кодын мөр бүр юу хийж байгааг тайлбарла.
Мөн эцэст нь дэлгэцэнд ямар хариу гарахыг ол.
START x = 5 x = x * x PRINT x END

➜ Мөр бүрийг тусад нь тайлбарлаж бич.

Бодлого 3
Компьютер хэрэглэгчээс нэг тоо асууна.
Тэр тооны куб зэргийг бодно.
(Тооны куб зэрэг гэдэг нь тухайн тоог өөрөөр нь 3 удаа үржүүлэх юм.)
Гарсан үр дүнг дэлгэцэнд харуулна.
