While давталт (Pseudocode)

1. While давталтын онол

While давталт нь өгөгдсөн нөхцөл үнэн (true) байх хугацаанд үйлдлүүдийг дахин дахин гүйцэтгэдэг.

Нөхцөл эхэнд шалгагддаг тул анхнаасаа худал байвал давталт огт ажиллахгүй.

2. While давталтын ерөнхий хэлбэр

WHILE (нөхцөл) DO
    үйлдлүүд
END WHILE

3. Жишээ 1: 1-ээс 5 хүртэл тоо хэвлэх

i = 1
WHILE (i <= 5) DO
    PRINT i
    i = i + 1
END WHILE

4. Жишээ 2: Эерэг тоо оруулах хүртэл асуух

INPUT x
WHILE (x <= 0) DO
    PRINT "Эерэг тоо оруулна уу"
    INPUT x
END WHILE

5. Дасгалууд

Дасгал 1. 10-аас 1 хүртэлх тоонуудыг буурахаар хэвлэх while давталтын pseudocode бич.

Дасгал 2. Хэрэглэгч 0 оруулах хүртэл оруулсан тоонуудын нийлбэрийг ол.

Дасгал 3. n тоо өгөгдөхөд 1-ээс n хүртэлх тэгш тоонуудыг хэвлэ.

6. Анхаарах зүйлс

  • Давталтыг зогсоох нөхцөл заавал хэрэгтэй
  • Хувьсагчийн утга давталт дотор өөрчлөгдөж байгаа эсэхийг шалга
  • Эс тэгвээс infinite loop үүснэ
