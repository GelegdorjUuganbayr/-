2.3 DATABASE

1. Database гэж юу вэ?

Database гэдэг нь мэдээллийг хүснэгт хэлбэрээр хадгалсан бүтэц юм.

Name Age City
Bat 12 UB
Sara 13 Darkhan

2. Field ба Record

Field = Багана (жишээ: Age)

Record = Мөр (жишээ: Bat – 12 – UB)

3. Data type

Data type = тухайн field ямар төрлийн мэдээлэл хадгалах вэ?

  • Number → 5, 12, 100
  • Text → Bat, UB, puzzle

4. Primary Key

Primary key = record бүрийг ялгах давтагдашгүй утга.

ID Name Age
1 Bat 12
2 Sara 13

ID нь primary key. Давтагдаж болохгүй.

5. Query

Query = мэдээлэл хайх дүрэм.

  • Age > 12
  • Name = "Bat"
  • City = "UB"
