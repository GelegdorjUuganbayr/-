Nested Loop – Давхар давталт

1. Давталтыг саная

Өмнөх хичээл дээр бид давталт (loop) үзсэн.

Давталт гэдэг нь нэг үйлдлийг олон удаа давтаж гүйцэтгэхийг хэлнэ. 

FOR i ← 1 TO 5
    PRINT i
NEXT i

Энэ код 1-ээс 5 хүртэлх тоог хэвлэнэ.

2. Давхар давталт гэж юу вэ?

Заримдаа бид давталтын дотор дахин давталт хийх шаардлагатай болдог.

Үүнийг Nested Loop (давхар давталт) гэдэг.

Гол ойлголт:
  • Гаднах loop → мөр
  • Дотор loop → багана

3. Энгийн жишээ

FOR row ← 1 TO 3
    FOR col ← 1 TO 4
        PRINT "*"
    NEXT col

    PRINT NEWLINE
NEXT row

Энэ кодын үр дүн:

****
****
****

Яагаад ингэж гарч байна вэ?

  • row loop → 3 удаа ажиллана
  • col loop → мөр бүр дээр 4 удаа ажиллана

4. Алхамчилсан ойлголт

Энэ кодыг харъя. 

FOR i ← 1 TO 4
    FOR j ← 1 TO i
        PRINT "*"
    NEXT j

    PRINT NEWLINE
NEXT i

Үр дүн:

*
**
***
****

Энд:

  • i → мөрийн дугаар
  • j → хэдэн од хэвлэхийг шийднэ

  • Nested loop = давталтын дотор давталт
  • Гаднах loop → мөр
  • Дотор loop → багана
